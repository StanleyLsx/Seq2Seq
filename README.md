# Seq2Seq

此仓库是基于Tensorflow官方教程的[基于注意力的神经机器翻译](https://tensorflow.google.cn/tutorials/text/nmt_with_attention)重构的。  
改造之后的模型将同时适用于机器翻译和摘要生成。可直接通过配置启动训练并提供推断功能。  


## 环境  
* pandas==1.2.4  
* tensorflow-gpu==2.3.0  
* tqdm==4.61.0  
* sklearn==0.0  
* jieba==0.42.1
